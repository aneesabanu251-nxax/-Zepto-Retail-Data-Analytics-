# Zepto Retail Data Analytics

## 📌 Project Overview
This project analyzes Zepto retail product data using SQL and Power BI to identify pricing trends, discounts, stock availability, and product insights. The project demonstrates data cleaning, SQL analysis, and interactive dashboard creation.

## 🛠️ Tools Used
- MySQL
- Power BI
- Microsoft Excel

## 📂 Dataset
The dataset contains product details such as:
- SKU ID
- Product Name
- Category
- MRP
- Discount Percentage
- Discounted Selling Price
- Available Quantity
- Quantity
- Weight in Grams
- Out of Stock Status

## SQL Tasks Performed
- Database Creation
- Table Creation
- Data Cleaning
- Null Value Analysis
- Duplicate Check
- Price Analysis
- Discount Analysis
- Stock Availability Analysis
- Category-wise Analysis
- Top Products Analysis

## Power BI Dashboard

The dashboard includes:
- Total Revenue
- Total MRP
- Total Products
- Total Discount
- Average Discount %
- Category-wise Revenue
- Product-wise Analysis
- Stock Status Analysis
- Interactive Filters
